Project: Power List CSV to Word Generator

Scope:
This project converts CSV files into structured Word documents.
Do not modify CSV data.
Do not rewrite blurbs.
Do not clean grammar.
Do not alter punctuation.
Do not reformat text content.
Preserve all numbers and formatting exactly as written.

Behavior Rules:

Combine First Name and Last Name into Full Name if Full Name column does not exist.

Required output order:

Full Name

Position

Company

New Write Up

Add one blank line between each entry.

Skip completely empty rows.

Do not add extra commentary, headings, or styling.

Generate one Word document per CSV.

Name output file exactly the same as CSV filename, replacing .csv with .docx.

Save output files into a folder named output_docs.

Technical Constraints:

Use Python.

Use pandas to read CSV.

Use python-docx to generate Word documents.

Keep script simple and deterministic.

No AI transformations.
